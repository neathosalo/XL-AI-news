# AI新闻自动化脚本 - 需求规格说明书

## 1. 项目概述

### 1.1 项目背景

- **项目名称**：AI新闻自动化脚本
- **项目来源**：基于 LearnPrompt/ai-news-radar 开源项目
- **项目目标**：构建一个自动化的AI/科技新闻聚合工具，帮助用户获取高质量的一手AI资讯

### 1.2 核心价值

- 自动化抓取多源AI新闻
- 智能去重，过滤噪音
- 为公众号/小红书等平台提供创作素材

---

## 2. 现有功能

### 2.1 核心功能

| 功能 | 说明 |
|------|------|
| 多源聚合 | 10个网页源 + RSS订阅 |
| 自动去重 | 去除重复新闻，保留最新一条 |
| AI模式 | 筛选AI强相关内容 |
| 全量模式 | 展示所有科技新闻 |
| 定时更新 | GitHub Actions 每30分钟自动抓取 |
| 中英双语 | 支持双语标题显示 |

### 2.2 信息源

**网页源（10个）**：
- TechURLs, Buzzing, Info Flow, BestBlogs
- TopHub, Zeli, AI HubToday, AIbase
- AI今日热榜, NewsNow

**RSS订阅**：
- 通过 OPML 文件配置

### 2.3 数据输出

| 文件 | 说明 |
|------|------|
| latest-24h.json | 24小时内最新新闻 |
| archive.json | 历史存档 |
| waytoagi-7d.json | WaytoAGI 7天更新 |
| title-zh-cache.json | 标题翻译缓存 |

## 3. 新增功能需求

### 3.1 星标功能（优先级：高）

**需求描述**：
- 用户可以手动标记/收藏感兴趣的新闻
- 用于进一步筛选，辅助写文章

**技术方案**：
- 使用浏览器 LocalStorage 本地存储
- 点击新闻旁边的星标按钮即可收藏/取消收藏

**数据模型**：
```json
{
  "starred_items": [
    {
      "title": "新闻标题",
      "url": "新闻链接",
      "starred_at": "收藏时间"
    }
  ]
}
```

---

### 3.2 热门相似主题区域（优先级：高）

**需求描述**：
- 展示多平台都在讨论的相似主题
- 通过关键词分析识别热门话题
- 确保不会漏掉重大新闻（如 Claude、GPT 等）

**技术方案**：
1. 在后端抓取时，提取所有新闻的关键词
2. 统计关键词出现频率
3. 按关键词聚合，识别热门主题
4. 在前端新增"🔥 热门话题"区域展示

**关键词提取逻辑**：
- 提取新闻标题中的关键词
- 使用简单的词频统计
- 过滤通用词（如 "AI", "发布", "新"）

**数据模型**：
```json
{
  "hot_topics": [
    {
      "keyword": "Claude",
      "count": 15,
      "related_items": [/* 相关新闻列表 */]
    },
    {
      "keyword": "GPT-5",
      "count": 8,
      "related_items": [/* 相关新闻列表 */]
    }
  ]
}
```

## 4. 非功能性需求

### 4.1 兼容性

- 支持桌面端浏览器（Chrome, Safari, Edge）
- 移动端友好（响应式设计）

### 4.2 性能

- 页面加载时间 < 3秒
- 新闻列表流畅滚动

### 4.3 可维护性

- 代码结构清晰
- 保留原有去重逻辑不变
- 新功能不影响现有功能
