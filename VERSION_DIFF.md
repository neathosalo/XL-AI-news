# 版本差异记录

本文档记录 `main`（GitHub公开版）与 `local`（本机版）的功能差异。

---

## 分支说明

| 分支 | 用途 | 说明 |
|------|------|------|
| `main` | GitHub公开版 | 分享给朋友和外部用户 |
| `local` | 本机版 | 个人自用，功能更丰富 |

---

## 功能差异

| 功能 | main (GitHub) | local (本机) | 备注 |
|------|---------------|--------------|------|
| 数据文件 (data/) | ❌ 不包含 | ✅ 包含 | 本机版有实时数据 |
| 星标收藏 | ❌ 无 | ✅ 有 | 前端LocalStorage存储 |
| 热门转载新闻 | ✅ 有 | ✅ 有 | 多平台转载识别 |
| 热门关键词 | ✅ 有 | ✅ 有 | 标题关键词提取 |
| WaytoAGI板块 | ❌ 无 | ✅ 有 | AI学习社区更新 |
| 访问量统计 | ✅ 有(不蒜子) | ❌ 无 | 仅GitHub版需要 |
| 说明文字 | "默认筛选AI/科技" | "附带WaytoAGI近7日更新" | 差异化描述 |
